Bienvenido al Objeto Virtual de Información (O.V.I.) del curso de Administración de redes, los contenidos presentados a continuación son herramientas desarrolladas para ayudar a la comprensión de las diferentes temáticas de esta temática.

Durante la exploración de este material encontrara: lecturas, videos y actividades de evaluación las cuales están diseñadas para obtener una adecuada apropiación sobre este tema.

OBJETIVO OVI

Desarrollar un objeto virtual de información (OVI), para el curso Administración de redes, el cual mediante el uso de diversos recursos informáticos sea una herramienta que permita al estudiante identificar amenazas de seguridad a las que se expone una red.

OBJETIVO CURSO

Presentar la información necesaria para el desarrollo de las tematicas del curso de administracion de redes, utlizando diferentes herramientas multimedia, para una mejor apropiacion del conocimiento requerido.
